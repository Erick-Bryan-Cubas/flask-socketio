Flask SocketIO

Este exemplo ilustra uma aplicação Flask utilizando Flask-SocketIO para comunicação em tempo real entre o servidor e o cliente através de WebSockets. Quando um usuário se conecta à aplicação, o cliente envia automaticamente uma mensagem "User has connected!" para o servidor via WebSocket.

Do lado do servidor, ao receber uma mensagem, ele responde enviando um elemento da lista pré-definida (contendo 'apple', 'peas', 'juice', 'orange') de volta para o cliente. Cada mensagem recebida resulta no envio do próximo item da lista, até que todos os elementos sejam enviados.

Este processo demonstra como estabelecer uma conexão bidirecional entre o cliente e o servidor, permitindo a troca de mensagens em tempo real. A aplicação utiliza eventos específicos para lidar com a conexão e a comunicação, o que é fundamental para aplicações interativas em tempo real.
